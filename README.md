# HUBOT Global 420 
A simple command for [Hubot](https://hubot.github.com/). It Answers

- Where is it going to be 4:20 PM next?
- How soon?

Based on 

https://github.com/thedod/global420

## Usage

`@robot 420`

## Install
Just copy *global420.js* and *helpers* directory to your *scripts directory*.
